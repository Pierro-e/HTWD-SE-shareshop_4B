== Einführung
Dieser Projektplan beschreibt das Software Engineering-Projekt „ShareShop“, eine kollaborative Einkaufsliste für Wohngemeinschaften. Ziel ist es, den Einkaufsprozess zu optimieren und Konflikte rund um Kostenaufteilung und Bedarfsplanung zu vermeiden. Das Dokument erläutert die Projektorganisation, verwendete Arbeitsmethoden und den Einsatz von Tools zur effizienten Zusammenarbeit.
