. lokalen "developer" Branch updaten
[source,bash]
git switch <local-developer-name>
git pull origin developer

. neuen "task" Branch erstellen
[source,bash]
git switch -b task/<name>
git push --set-upstream <remote-repo> task/<name>

. lokales Arbeiten und Dokumentieren auf lokalem "task" Branch
[source,bash]
git add ...
git commit ...

. in lokalen "developer" Branch mergen
[source,bash]
git switch <local-developer-name>
git pull
git merge --no-ff task/<name>

. testen und Fehler beheben
. Pull-Request in GitHub erstellen
. Feedback aus Pull-Request umsetzen
. Pull-Request schließen

. remote "developer" Branch updaten
[source,bash]
git push

. aufräumen
[source,bash]
git push origin -d task/<name>
git branch -d task/<name>
